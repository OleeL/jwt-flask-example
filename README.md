# JWT example project Flask Python

Just a basic set up for a JWT project in python

## Mac setup:

### Create env
```bash
python3 -m venv env
```

### Activate env
```bash
source env/bin/activate
```

### Decactivate env
```bash
deactivate
```

## Dependencies

```bash
pip install flask
pip install PyJWT
pip install python-dotenv
```

## Environments
```bash
cp .env.template .env
```

## Running the project
Run with:
```bash
python main.py
```
